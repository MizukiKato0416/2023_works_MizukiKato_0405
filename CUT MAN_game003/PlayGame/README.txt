//ゲームタイトル
CUT MAN

//制作期間
2020年12月〜2021年2月　(約2ヶ月)

//開発環境
Visual Studio 2015　,　C/DirectX9

//制作人数
1人

//ルール
制限時間内に多くの敵を倒してスコアを稼ぎ、ランクインを目指します。
攻撃方法は弱攻撃、強攻撃、全体攻撃の合計3つ。
弱攻撃は威力が低いものの隙が少なく、1つ目の斬撃を出している最中にもう1度弱攻撃ボタンを押すことで連続で攻撃ができます。
強攻撃は隙が大きいですがその分威力も高くヒットの持続時間が長い。命中することで敵をノックバックさせることができます。
全体攻撃は360度全方向に対して斬撃を放つことができます。威力は弱いですが命中した敵をノックバックさせることができます。
ガードボタンを押すことで全方位の敵の攻撃をガードすることができます。
プレイヤーのHPゲージが0になったらゲームオーバー。

//操作方法
XInputのゲームパッドでのみ遊ぶことができます。
LStick	 ：プレイヤー移動／カーソル移動
RStick	 ：視点移動
LT／7	 ：視点位置リセット
RT／8	 ：ガード(長押し)
START	 ：ポーズ／ポーズ解除／次に進む
A		 ：戻る
B		 ：次に進む／弱攻撃／決定
Y		 ：強攻撃
X		 ：全体攻撃
ESC	  	 ：ゲームを終了する

//アピールポイント
初めての3Dゲーム作品です。授業内では学習していない部分として、プレイヤーの移動を入力したときやカメラの視点を
リセットするとき、敵がプレイヤーの方向を向くときに滑らかに回転するようにしました。
また、プレイヤーと敵との距離が一定の距離に達した際、敵が自動的にプレイヤーの方向を向いて追従するように
プログラムしました。
